# 设计模式

## 设计模式简介



### 常见的设计模式

常见的18种设计模式如下：  

- Abstract Factory 抽象工厂
- Builder 生成器
- Factory Method 工程方法
- Prototype 原型
- Singleton 单件
- Adapter 适配器
- Bridge 桥接
- Composite 组合
- Decorator 装饰
- Facade 外观
- Flyweight 享元
- Proxy 代理
- Chain of Responsibility 指责链
- Command 命令
- Interpreter 解释器
- Iterator 迭代器
- Mediator 中介者
- Memento 备忘录
- Observer 观察者
- State 状态
- Strategy 策略
- Template Method 模版方法
- Visitor 访问者
